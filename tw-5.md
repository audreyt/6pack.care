交通更安全不是因為每個駕駛都是聖人，而是因為道路、標誌和規則使安全駕駛比魯莽駕駛更容易。

即使完美的本地關懷，在敵對的更廣泛生態系統中也會失敗。團結力裝備整個領域，使公民行為透過設計獲勝。

特龍托的第五階段——她在《關懷民主》中的新增——要求「關懷需求及其被滿足的方式，必須與對正義、平等和所有人自由的民主承諾保持一致。」這需要「多元性、溝通、信任和尊重。」這是關懷的政治肌肉：不僅是服務個人，而是確保系統本身是公平的。

### 收回免責牌

團結力意味著沒有人能獲得免除關懷的結構性豁免。特龍托的比喻：「我們首先需要做的，是收回所有那些免除認真承擔關懷責任的免責牌。沒有人因為從事保護、生產或照顧自己人，或者富裕到可以自力更生或捐款慈善，就自動獲得免除關懷的免責牌。」

對 AI 系統而言，等同的免責牌是：「我們太大了不能被監管」、「用戶在服務條款中同意了」、「我們開源了權重」、「我們捐款給安全研究了」。團結力透過使合作在結構上無法迴避，來收回這些免責牌。

### 核心理念

- **無曝光的身份。** 代理 ID 斷言「這個代理錨定到人類/法人實體」而不公開揭示該人（部分匿名性）。
- **互通性勝於平台鎖定。** 可攜性和開放協議將競爭轉移到關懷品質。用戶以標準格式（ActivityPub/AT 類）匯出社交圖譜，離開時保留觀眾。
- **聯邦架構勝於單體架構。** 分享威脅情報而不需受制於單一瓶頸。網路透過開放聯盟分享文本威脅情報和嵌入指紋；本地政策保持本地，而防禦複合。
- **言論自由不等於觸及自由。** 推薦系統問責是公民義務。搭橋式排序獎勵增加跨群體認可的內容；只煽動單一群集的內容不獲演算法助力。

### 良好的團結力是什麼樣子

- **代理 ID 登記處（部分匿名性）。** 代理（人類/組織/AI）具有由受信任託管者持有的證明；公開證明是最小的但可驗證的。
- **社交可攜性。** 用戶匯出社交圖譜和內容；通過互通性測試；離開時保留觀眾。猶他州數位選擇法案（2025 年生效）是一個運作中的原型。
- **搭橋透明性。** 平台公布搭橋指數、跨群體認可分數和失利下的信任審計。
- **聯邦信任與安全。** 如同 ROOST 的 CSAM 防禦：合作夥伴訓練本地 AI 在其特定文化脈絡中偵測危害，透過聯邦學習分享威脅情報，執行保持本地化。
- **協議層級規範。** 機器可讀的合作條款：未經同意不抓取；尊重申訴 webhook；尊重退出。

### 從理念到實踐

1. **建立代理 ID 託管者。** 社區組織或公共利益實體發行/持有證明；公布撤銷和質疑端點。
2. **在採購中強制要求可攜性。** 公共買家要求協議互通和帶著信任退出演練。
3. **採納搭橋審計。** 平台每季度公布關係健康指標；第三方驗證。
4. **加入安全聯盟。** 貢獻並取用共享威脅登記處的資訊；本地化執行。
5. **預設搭橋訂閱源。** 使搭橋式排序成為公民背景的預設。

### 工具（今天就能建立）

- **代理 ID 模式**（DID/VC、撤銷清單、證明格式）。
- **可攜性工具**（匯出/匯入腳本；資料保真度檢查）。
- **搭橋審計套件**（計算並公布重疊指標）。
- **聯邦 T&S 中心**（威脅情報的開放 API；本地適配器）。
- **條款即程式碼**（代理可以自動檢查的政策模式）。

### 洪水聊天機器人故事：第五部

- **代理 ID。** 志願翻譯員和洪水聊天機器人註冊部分匿名 ID；濫用下降；問責上升而不曝光私人資訊。
- **可攜性。** 倖存者將案件檔案和聯絡人從緊急機器人移動到長期住房服務，無需重新輸入一切。
- **聯邦防禦。** 來自鄰近城市的詐騙報告透過安全中心流入；機器人預設降級可疑連結。
- **搭橋指標。** 每週公開搭橋指數顯示租房者和房主共同認可的提案比第一週更多。

### 可能出錯的地方

- **ID 蔓延。** ID 變成檔案。解決：設計上的部分匿名性；最小證明；獨立託管者；強撤銷。
- **可攜性作秀。** 匯出不可讀或有損。解決：契約中的退出測試；保真度失敗的懲罰。
- **聯邦俘獲。** 一個大玩家決定規範。解決：多中心治理；開放標準；輪換管理者。
- **聯邦即攻擊面。** 沒有安全加固，一個本地代理網路只是一個等待被招募的去中心化殭屍網路。解決：在每個節點強制要求沙盒、最小權限執行和輸入驗證（Pack 3）；聯邦安全審計；共享漏洞揭露。

### 與其他「力」的介接

- **來自回應力（Pack 4）：** 修復文化餵養跨組織信任。
- **來自負責力（Pack 2）：** 可攜性和退出條款在每份參與契約中被引用。
- **對共生力（Pack 6）：** 團結力提供本地地神合作所需的條約。

### 結語意象：標示清楚的交流道

想像一個標示清楚的交流道——多條車道、明確的標誌、安全的匯流——行駛更順暢，因為道路是為共享而建的。
