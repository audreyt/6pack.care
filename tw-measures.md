每個「力」回答關懷循環中一個獨立的階段問題。以下指標的分配方式確保每個指標只歸屬於一個「力」——互斥且窮盡。

### Pack 1——覺察力：我們是否看對了方向？

| 指標       | 回答什麼問題                                                     |
| ---------- | ---------------------------------------------------------------- |
| 覆蓋率     | 受影響群體中有多少比例提供了輸入？                               |
| 缺席率     | 哪些人口群體在搭橋地圖中沒有聲音？                               |
| 聲音公平性 | 最少被聽到的群體與最多被聽到的群體，分別獲得多少引導時間？       |
| 搭橋地圖   | 哪些利害關係群體被涵蓋？哪些裂縫是相互強化的、哪些是交叉切割的？ |

### Pack 2——負責力：我們是否做出並信守了正確的承諾？

| 指標         | 回答什麼問題                                                   |
| ------------ | -------------------------------------------------------------- |
| 承諾覆蓋率   | 已辨識出的需求中，有多少比例有具名負責人和 SLA？               |
| SLA 遵循率   | 案件在其嚴重性類別的時限內解決的比例是多少？                   |
| 採納或解釋率 | 公民大會產出中，有多少比例有記錄的採納或公布的偏離說明與補救？ |

### Pack 3——勝任力：我們是否正確執行？

| 指標         | 回答什麼問題                                                     |
| ------------ | ---------------------------------------------------------------- |
| 決策準確度   | 按嚴重性類別的錯誤率是多少？                                     |
| 防護網完整性 | 紅線測試通過的比例是多少？                                       |
| 追蹤完整性   | 具有完整追蹤記錄（規則、來源、不確定性、收據）的決策比例是多少？ |
| 金絲雀健康度 | 金絲雀發布中未觸發轉返而完成的比例是多少？                       |

### Pack 4——回應力：關懷是否妥善到位？

| 指標         | 回答什麼問題                                   |
| ------------ | ---------------------------------------------- |
| 解決率       | S0/S1 案件在承諾時限內成功解決的比例是多少？   |
| 申訴結案時間 | 按嚴重性類別，實際結案時間與目標時間相比如何？ |
| 傷害復發率   | 已解決的事件在 90 天內再次出現的比率是多少？   |
| 失利下的信任 | 發生不良結果後的信任分數是多少——修復是否奏效？ |

### Pack 5——團結力：生態系統在結構上是否公平？

| 指標             | 回答什麼問題                                       |
| ---------------- | -------------------------------------------------- |
| 搭橋指數         | 共同決策中跨群體參與和認可的比率是多少？每季公布。 |
| 可攜率           | 離開時成功匯出資料的使用者比例是多少？             |
| 智慧體 ID 覆蓋率 | 擁有可驗證部分匿名證明的智慧體比例是多少？         |

### Pack 6——共生力：系統是否有界且可持續？

| 指標       | 回答什麼問題                                       |
| ---------- | -------------------------------------------------- |
| 範圍合規率 | 智慧體行動落在宣告目的界限內的比例是多少？         |
| 繼任準備度 | 上次成功的退出演練是什麼時候？                     |
| 退場合規率 | 擁有有效證明和啟用中退場計時器的智慧體比例是多少？ |
| 生態多樣性 | 在同一領域中提供相同需求服務的獨立智慧體有多少？   |

### 信任的分解

信任貫穿所有六個階段，但每個「力」衡量的是一個不同的維度：

| Pack      | 信任維度                                 |
| --------- | ---------------------------------------- |
| 1——覺察力 | 被傾聽的信任（聲音公平性）               |
| 2——負責力 | 對承諾的信任（SLA 遵循率、採納或解釋）   |
| 3——勝任力 | 對執行的信任（追蹤完整性、防護網完整性） |
| 4——回應力 | 傷害後的信任（失利下的信任）             |
| 5——團結力 | 跨群體的信任（搭橋指數）                 |
| 6——共生力 | 跨時間的信任（繼任準備度、範圍合規率）   |

**S0–S3** 是在 Pack 2 中定義的嚴重性分類器（S0 生命/安全；S1 生計；S2 便利；S3 使用者體驗）。它在 Packs 3、4 和 6 中作為通用量尺使用，但本身不是一個衡量指標。

**跨群體認可**以兩種不同角色出現：在 Pack 4 中作為 RLCF 訓練信號（訓練目標——如何塑造模型），以及在 Pack 5 中作為搭橋指數的基礎（生態系統稽核——對外報告的內容）。這是兩種不同的用途。
