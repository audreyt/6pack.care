# 6-Pack of Care

[6pack.care](https://6pack.care/) is a bilingual (British English / Traditional Mandarin) static site presenting the "6-Pack of Care" research framework on AI ethics and alignment by Audrey Tang and Caroline Green at Oxford's Institute for Ethics in AI. Licensed CC0 (public domain).
